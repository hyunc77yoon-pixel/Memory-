✅ 기술적으로 의미 있는 부분
τ = RC를 스케일링 지표로 삼는 것은 회로 이론적으로 정확. DRAM word line delay, bit line RC, sense amp 타이밍 모두 RC 지배. 1.5µm pitch HB는 공정상 가능한 범위. 수출 규제 우회 전략(EUV 불필요)으로서 CXMT/YMTC 방향과 일치.
⚠️ 검증 필요 항목
• 238 MTr/mm² 비교 기준 미공개 (TSMC N3E ~170 MTr/mm², N2 ~200 MTr/mm²와 비교 방법 불명)
• "1.4nm 동등"은 공정 노드가 아닌 밀도 기준 — 혼동 주의
• 1.5µm pitch HB 수율·신뢰성 데이터 없음 (HBM3E die-to-die ~9µm, HBM4 목표 ~3-4µm 대비)
• Kirin 2026 출하 전 수치 — 양산 결과 독립 검증 필요
📌 DRAM 설계에 주는 시사점
• 메모리 접근 지연이 τ의 칩 레벨 핵심 축으로 명시 → Logic-DRAM 3D 통합 가속 시사
• UnifiedBus의 Native Memory Semantics = DRAM bandwidth·latency가 시스템 병목 → HBM 수직 통합 방향 확정
• 화웨이 자체 HBM 1.6 TB/s 목표 → CXMT HBM3 로드맵과 독립적 경쟁 구도
• 6년간 381개 칩 τ 적용 → VLSI 2022 IGZO 3D DRAM과 연속선상, 메모리+로직 3D 통합 내부 축적 증거
